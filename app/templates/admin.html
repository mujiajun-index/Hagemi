API 代理路由映射 ▶

功能描述: 此功能允许您将特定的 API 请求前缀代理到不同的目标地址。例如，您可以将所有发往 /openai 的请求转发到 https://api.openai.com。

使用示例: 直接访问本项目地址 http://localhost:7860/openai/v1/chat/completions 即可将请求代理到 https://api.openai.com/v1/chat/completions
